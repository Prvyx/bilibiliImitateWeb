#### 项目名称：react-bilibili
#### 需求分析
- 模仿b站，实现b站核心功能
  - 页面的布局（react组件，spa）
    - 首页
    - 番剧
    - ...
    - 布局的设计思路：根据对b站的ob可知，上述的大部分网页的共同组件有：
      - 导航栏（TopMenu组件）
      - 分类栏（Catalog组件）
      - 底部栏（Footer组件）
    - 所以，将这三个组件放在App.js中，中间的内容部分根据路由的不同进行相应的展示
#### 详细设计
- 依赖react的组件库进行设计
#### 功能：
- 做一个模仿小破站的项目，人们可以做uploader，也可以做观看者。喜爱分享

- react：
  - 采用create-react-app脚手架快速搭建react框架
